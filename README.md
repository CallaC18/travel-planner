# Travel Budget Planner

一个用于旅行攻略规划的 Codex skill，并附带一个本地网页交互界面。

## 文件说明

- `SKILL.md`：Codex skill 主说明文件
- `index.html`：网页交互界面
- `styles.css`：彩色界面样式
- `app.js`：表单交互和攻略草稿生成逻辑

## 如何打开界面

直接双击 `index.html`，或在浏览器中打开它即可。

## 如何使用

1. 填写目的地、人数、日期、出发地等基础信息。
2. 勾选景点，选择酒店和航班偏好。
3. 点击“生成攻略草稿”。
4. 把生成的草稿复制给 Codex，让它联网复核并生成最终攻略。

## 推荐流程

完整攻略输出后，先确认是否满意；满意后再导出 `.docx` 攻略文档。
